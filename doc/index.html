cardEngine overivew
Right now there are 2 parts of Card Engine.
stage 1: Solitaire (Klondike)
stage 2: Oasis (hearthstone clone)





Solitaire

Solitaire is a simple Klondike clone.
Click on a card to select it.
Click on the card stack in the top left to flip them over one by one.
When you have the next card for the foundation (starting at ace, raising up till king) click on the foundation pile in the bottom center to move the card there.


Render: solitaire_render() Event: solitaire_event()



Oasis


Oasis is a simple Hearthstone style clone, specifically a clone of "The League of Explorers" expansion.
Where you construct your deck by selecting your hero, their power, starting packs, and then between each combat you:
expand you deck by adding packs of three cards,
visit the shop to modify existing cards,
add board / character wide passives.


Render: oasis_render() Event: oasis_event()

Targeting // 0 is targets allies // 1 enemies // 2 aoe all minion // 3 aoe allies minions // 4 aoe enemy minions // 5 is hit aly champing // 6 is hit enemy champing // 7 is AOE all (minions and champs) // 8 is AOE all enemies (minions and champ) // 9 is AOE all allies (minions and champ) enum target_side { tar_ally = 0, tar_enemy, tar_either, }; enum target_count { target_everyone = 0, target_num, // to hit 1 person, do = target_num, to hit 2 people, do target_num+1, do i ever want to target 0 people? }; enum target_who { tar_champ = 0, tar_minion, tar_both, tar_rand, }; // if i want to target the ally minions + champ, along with only enemy minions, i cant do that rn.


Effect // 0 is dmg, // 1 is heal // 2 is apply modifier // 3 is draw x cards // 4 is summon a specific card / random card / pseudo random, idk.


Example:
"Hat", gives +2/+2 to a friendly minion, and is shuffled back into the deck after being played.
It will have the target:
side = aly
who = minion
it will have 2 effects, one that gives +2 atk, the other +2 hp.
Even though there are 2 effects, there is only one "modName" that will show up at the bottom of the minion card.
the discard is: discard_shuf_playDeck;

base = make_cardBase_spell ( "Hat" ); base->card->mana = 1; spell = base->card->spell; strcpy ( spell->modName, "Hat: +2/+2" ); spell->discard = discard_shuf_playDeck; tar = spell->tar; tar->side = tar_aly; tar->who = tar_minion; tar->count = target_num + 0; // casts on 1+0 guy(s) eff = spellEffectInit ( ); arrayListAddEndPointer ( spell->effList, eff ); eff->type = eff_buffHp; eff->heal = 2; eff = spellEffectInit ( ); arrayListAddEndPointer ( spell->effList, eff ); eff->type = eff_buffAtk; eff->heal = 2;
